STEMeet

STEMeet is a social platform built to connect students, professionals, and STEM enthusiasts. Developed for Lab 2 course using Django REST Framework, React.js, MySQL, and JWT authentication. Key features include an AI Assistant powered by OpenAI GPT-4o mini API for scientific questions and project suggestions, real-time communication with modern chat and video calls via WebSockets, professional networking to find and add friends, collaborative project management with role-based access, and advanced personal profiles showcasing experience and contributions to the STEM community. View on GitHub.

AcademixPro

AcademixPro is a school management system built with Laravel, React, and MySQL, designed to streamline academic operations for both students and professors. The platform provides an intuitive interface for managing academic activities efficiently. For students, it offers personalized dashboards to view grades and GPA, access lecture schedules, choose study groups, and register for semesters. For professors, it provides tools for grading students, adding lectures, and recording attendance. AcademixPro combines functionality with simplicity, ensuring a centralized and reliable system for academic workflows. View on GitHub.

LuxuryStep

LuxuryStep is an e-commerce platform for luxury shoe reviews and sales developed for the Software System Design course. Users can browse products, add shoes to wishlist, read and write reviews, and make online purchases with Stripe API integration. Built with React.js frontend, Django REST Framework backend, and SQLite database. The project includes comprehensive technical documentation with conceptual models, class diagrams, and low-level design specifications. View on GitHub.

Transify

Transify is a comprehensive transportation management platform built with ASP.NET Core 8.0 and Entity Framework. The system provides unified management for both bus and taxi services with multi-role dashboards for administrators, business owners, and drivers. Key features include real-time booking and reservation management, route planning, fleet coordination, company management support, and role-based access control. The platform offers search and filter capabilities, business analytics, review systems, and a responsive Bootstrap interface for seamless user experience across all devices. View on GitHub.

Bird Tracker

A mobile application for bird watching enthusiasts to track and identify different bird species. Includes GPS location tracking, photo capture, and a comprehensive bird database with detailed information. View on GitHub.

Stock Prediction

An LSTM-based machine learning project that predicts NVIDIA stock prices using historical data. The system collects data through yfinance, processes it for training, and uses the trained model to forecast future stock prices. The model's accuracy is evaluated by comparing predicted prices with actual prices during the testing period, demonstrating practical application of deep learning in financial forecasting. View on GitHub.

Image Classifier

A deep learning image classification project using the CIFAR-10 dataset, which contains 10 object categories including airplanes, cars, animals, and more. The system normalizes images for optimal model performance and uses a pre-trained image_classifier.keras model to classify new images. The project demonstrates practical computer vision applications with real-time image processing and classification capabilities. View on GitHub.

Urban Sips

Urban Sips is a React web application that helps users discover coffee shops, cafes, and bars in Prishtina and all of Kosovo. The app features an interactive map with venue locations, search functionality for specific venues by name, and marker clustering to keep the map clean and organized. Built with React, Leaflet maps, and marker clustering for optimal performance, the application uses OpenStreetMap's Overpass API to fetch real-time venue data based on the current map area, providing users with up-to-date information about local establishments.
